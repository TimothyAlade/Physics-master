Chapter 3: Scalars and Vectors

📘 SS1 First Term ⏱️ Reading: 30 mins

🎯 Learning Objectives

  • Define scalar and vector quantities
  • Give examples of scalars and vectors
  • Represent vectors graphically using arrows
  • Add and subtract vectors using graphical methods
  • Resolve vectors into perpendicular components
  • Apply vector concepts to solve physics problems

3.1 What are Scalars and Vectors?

📌 Scalar Quantity: A quantity that has magnitude only (size or numerical value).
Examples: Mass (5 kg), Time (10 s), Distance (100 m), Speed (60 km/h), Temperature (25°C), Energy (500 J).
📌 Vector Quantity: A quantity that has both magnitude and direction.
Examples: Displacement (5 m North), Velocity (60 km/h East), Force (10 N downward), Acceleration (9.8 m/s² downward).
💡 Memory Tip: "Scalar = Scale only" (just the number). "Vector = Direction matters."

3.2 Representation of Vectors

Vectors are represented graphically by arrows. The length of the arrow represents the magnitude, and the arrowhead shows the direction.

Vector representation: An arrow showing magnitude and direction

In written form, vectors are represented by:

  • Bold letters: F (force), v (velocity), a (acceleration)
  • Letters with an arrow above: F⃗, v⃗, a⃗
  • Underlined letters: F, v, a

3.3 Vector Addition

Vectors are added differently from scalars. You cannot simply add the numbers—you must account for direction!

Graphical Method: Head-to-Tail

  1. Draw the first vector to scale in its given direction
  2. Draw the second vector starting from the head (arrow tip) of the first vector
  3. The resultant vector is drawn from the tail of the first vector to the head of the last vector
  4. Measure the length and direction of the resultant
📐 Example 1: Add a vector of 5 N East and 3 N East.
Since both are in the SAME direction, simply add the magnitudes:
Resultant = 5 N + 3 N = 8 N East
📐 Example 2: Add a vector of 5 N East and 3 N West.
Since they are in OPPOSITE directions, subtract the magnitudes:
Resultant = 5 N - 3 N = 2 N East

Parallelogram Method (Premium)

The parallelogram method is used when two vectors act from the same point but at an angle to each other.

  1. Draw both vectors from the same starting point (tail-to-tail)
  2. Complete the parallelogram by drawing parallel lines
  3. The resultant is the diagonal from the common starting point

Pythagorean Theorem for Perpendicular Vectors

R = √(A² + B²)

Example: A force of 3 N East and 4 N North act on an object. Find the resultant force.

R = √(3² + 4²) = √(9 + 16) = √25 = 5 N
Direction: θ = tan⁻¹(4/3) = 53.1° North of East

Cosine Rule for Non-Perpendicular Vectors

R² = A² + B² + 2AB cos θ

3.4 Vector Subtraction

To subtract vector B from vector A:

A - B = A + (-B)

This means: Reverse the direction of B, then add to A using the head-to-tail method.

📐 Example: Subtract 3 N West from 5 N East.
A = 5 N East
B = 3 N West
-B = 3 N East (reverse direction)
Resultant = 5 N East + 3 N East = 8 N East

3.5 Resolution of Vectors

Resolution means breaking a single vector into two perpendicular components (usually horizontal and vertical).

If a vector F makes an angle θ with the horizontal:

Fx = F cos θ (Horizontal Component)
Fy = F sin θ (Vertical Component)
📐 Example: A force of 20 N acts at 30° to the horizontal. Find its components.
Fx = 20 × cos(30°) = 20 × 0.866 = 17.32 N (horizontal)
Fy = 20 × sin(30°) = 20 × 0.5 = 10 N (vertical)

Finding Resultant from Components (Premium)

If you know the horizontal (Fx) and vertical (Fy) components:

Magnitude: F = √(Fx² + Fy²)
Direction: θ = tan⁻¹(Fy / Fx)

Practice Problems (Premium)

Q1: A force of 50 N acts at 60° to the horizontal. Find components.

Solution: Fx = 50 × cos(60°) = 25 N, Fy = 50 × sin(60°) = 43.3 N

Q2: Two forces of 6 N and 8 N act at right angles. Find the resultant.

Solution: R = √(6² + 8²) = √100 = 10 N, θ = tan⁻¹(8/6) = 53.1°

📝 Chapter 3 Summary

  • Scalars have magnitude only (mass, time, distance, speed)
  • Vectors have both magnitude and direction (displacement, velocity, force)
  • Vectors are represented by arrows (length = magnitude, arrowhead = direction)
  • Vector Addition: Head-to-tail method or parallelogram method
  • Vector Subtraction: A - B = A + (-B)
  • Resolution: Breaking a vector into components: Fx = F cos θ, Fy = F sin θ
  • Resultant from Components: F = √(Fx² + Fy²), θ = tan⁻¹(Fy/Fx)
