Chapter 7: Simple Machines

📘 SS1 First Term ⏱️ Reading: 35 mins

🎯 Learning Objectives

  • Define simple machines and their importance
  • Identify the six types of simple machines
  • Define and calculate Mechanical Advantage (MA)
  • Define and calculate Velocity Ratio (VR)
  • Define and calculate Efficiency
  • Solve problems involving simple machines

7.1 What are Simple Machines?

📌 Simple Machine: A basic mechanical device that changes the direction or magnitude of a force to make work easier.

Simple machines help us by:

  • Multiplying force (e.g., lever, pulley)
  • Changing direction of force (e.g., pulley)
  • Increasing speed or distance (e.g., gears)

7.4 Key Terms in Simple Machines

Mechanical Advantage (MA)

📌 Mechanical Advantage: The ratio of Load (output force) to Effort (input force).
It has NO UNITS (dimensionless).
MA = Load / Effort
📐 Example: A lever lifts a 500 N load with 100 N effort. Find MA.
MA = 500 / 100 = 5 (The machine multiplies force by 5)

Velocity Ratio (VR)

📌 Velocity Ratio: The ratio of distance moved by Effort to distance moved by Load.
It has NO UNITS (dimensionless).
VR = Distance moved by Effort / Distance moved by Load
📐 Example: Effort moves 50 cm, Load moves 10 cm. Find VR.
VR = 50 / 10 = 5

Efficiency (η)

📌 Efficiency: The ratio of useful work output to total work input, expressed as a percentage.
Efficiency = (MA / VR) × 100%
Efficiency = (Work Output / Work Input) × 100%
📐 Example: A machine has MA = 4 and VR = 5. Find efficiency.
Efficiency = (4 / 5) × 100% = 80%
📊 Important Relationships:
• For ideal machine (no friction): MA = VR, Efficiency = 100%
• For real machines: MA < VR, Efficiency < 100% (due to friction)
• Work Output = Work Input × Efficiency

7.5 Formulas for Specific Machines

MachineVR Formula
LeverVR = Effort Arm / Load Arm
Pulley SystemVR = Number of rope segments supporting load
Inclined PlaneVR = Length of plane / Height of plane
WedgeVR = Length / Thickness
ScrewVR = 2πR / Pitch
Wheel and AxleVR = Radius of Wheel / Radius of Axle
GearsVR = Number of teeth on driven gear / Number of teeth on driver gear

7.6 Work and Simple Machines

📌 Work: Force × distance moved in the direction of force.
Work Input = Effort × Effort Distance
Work Output = Load × Load Distance

Law of Conservation of Energy: A machine cannot create energy. Work output is always less than work input due to friction.

📐 Example: A pulley lifts 200 N load by 2 m using 50 N effort moving 10 m. Find work input, work output, and efficiency.

Work Output = 200 × 2 = 400 J
Work Input = 50 × 10 = 500 J
Efficiency = (400/500) × 100% = 80%

📝 Chapter 7 Summary

  • Simple Machines: Lever, Pulley, Inclined Plane, Wedge, Screw, Wheel & Axle
  • MA = Load / Effort (force multiplication)
  • VR = Effort distance / Load distance
  • Efficiency = (MA/VR) × 100%
  • Ideal machine: MA = VR, Efficiency = 100%
  • Real machine: MA < VR, Efficiency < 100% (due to friction)
